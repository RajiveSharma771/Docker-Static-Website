# Docker Static WebApp on AWS

## Project Overview
Beginner-friendly DevOps project to deploy a static website using Docker on AWS EC2.

## Tech Stack
- AWS EC2
- Docker
- Nginx
- Linux
- GitHub

## Architecture
Local → GitHub → AWS EC2 → Docker → Nginx

## Steps to Run
1. Launch Ubuntu EC2
2. Install Docker
3. Clone repository
4. Build Docker image
5. Run Docker container
6. Access via EC2 public IP

## Output
Static website successfully running inside Docker container on AWS EC2.
